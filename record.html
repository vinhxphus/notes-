🎙️ Ghi âm nhanh

Không rảnh gõ chữ thì ghi âm lại, lưu file rồi đính kèm vào note sau.
00:00
⬇️ Tải xuống file ghi âm
  1. Bấm "Bắt đầu ghi", cho phép trình duyệt dùng micro.
  2. Nói xong bấm "Dừng ghi", nghe lại thử ở khung phát bên trên.
  3. Bấm "Tải xuống file ghi âm" để lưu vào máy/điện thoại.
  4. Vào trang /admin, tạo hoặc sửa 1 note, đính kèm file vừa tải vào ô "Ghi âm".
